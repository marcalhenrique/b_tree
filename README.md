# BTree

## Descrição
Implementar uma BTree em C++.

## Rodar o Projeto
Instale as dependências:
```bash
sudo apt update
sudo apt install g++
sudo apt install build-essential
```
Depois compile o projeto utilizando o arquivo run.sh:
```bash
./run.sh
```
Esse arquivo compila, executa e limpa o projeto.
